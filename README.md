# 🧠 Cyber Security Training Game Platform

A modern interactive cybersecurity learning website built with **HTML, CSS, and JavaScript**.  
The project turns traditional learning into a **game-like experience** with quizzes, missions, and threat detection challenges.

---

## 🚀 Features

### 🧪 Cyber Quiz System
- 50+ cybersecurity questions
- Instant correct / wrong feedback
- Final score calculation
- Restart option

### ⚠️ Threat Detection Game
- 16+ cyber threat scenarios
- Risk level selection (Low / Medium / High)
- Score system (+10 / -5 logic)
- Locked answers after selection

### 🧠 Cyber Training Hub (Missions)
- Interactive learning missions
- Click-to-expand explanations
- Progress bar system
- XP gain system

### 💡 Fun Fact System
- Random cybersecurity facts
- Changes dynamically during interaction

---

## 🎮 Game Mechanics

- ✔ XP system for correct answers
- ✔ Score tracking across games
- ✔ Locked choices to prevent spam clicking
- ✔ Interactive learning instead of passive reading
